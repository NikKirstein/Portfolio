### TOBI
Traffic Obfuscated Internet.  A proof of concept project involving the mitmproxy python framework.  The project currently intercepts and stores HTML headers and then injects them back into requests along with modifying JavaScript.  The JavaScript currently has placeholder values but successfully has fed fake fingerprinting values to browserleaks.  Possible future development.
